## Initial Setup

Install updates. 

### Dock
* Enable magnification.
* Position Right.
* Auto hide/show dock.

### Trackpad
* Set "Tap to click" on Trackpad. 
* Turn off "natural" scroll direction.
* Set "Swipe between pages" to three fingers. 
* Increase trackspeed.
